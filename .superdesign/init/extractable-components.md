# Extractable Components — Alex Carpenter Studio

These are planned reusable patterns identified from the design spec.

## Navigation
- **SiteHeader** — sticky transparent→solid header with logo + nav links + mobile menu trigger
- **SiteFooter** — dark footer with logo, social links, copyright
- **MobileDrawer** — slide-in right overlay nav for mobile

## Homepage
- **HeroSplit** — full-viewport split layout: image left, content+CTA right
- **SectionTitle** — consistent section heading treatment (uppercase, letter-spaced)
- **FeaturedProjectCard** — thumbnail + title + client, hover zoom

## Portfolio
- **FilterBar** — horizontal pill/button filters: All, Video, Graphic Design, Web Design
- **PortfolioCard** — 16/9 thumbnail, title, client, category tag, hover overlay
- **ProjectHero** — full-width hero image for detail pages

## Services
- **ServiceCard** — icon + title + description + link, three-column grid

## UI Primitives
- **Button** — variants: filled (dark), outlined (dark), text link
- **AnimatedSection** — scroll-triggered fade-in/slide-up wrapper
- **VideoEmbed** — responsive 16/9 iframe wrapper

## Contact
- **ContactForm** — Name + Email + Message + Submit, with validation states
